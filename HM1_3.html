What is HTML?

HTML stands for HyperText Markup Language. It is the standard markup language used to create web pages. HTML markup consists of a series of tags which are used to structure the content and provide formatting.

Basic Structure of an HTML Page

An HTML page is made up of various elements like headings, paragraphs, links, images, etc. These elements are enclosed within opening and closing tags. For example, a heading can be created using the <h1> to <h6> tags.

Logical Formatting

In addition to physical formatting tags like headings and paragraphs, HTML also provides logical formatting tags. These tags are used to define the meaning of specific parts of the content. For example, the <abbr> tag is used to mark up an abbreviation, the <code> tag is used to mark up computer code, and the <dfn> tag is used to mark up a definition.

        <p>The <abbr title="HyperText Markup Language">HTML</abbr> element is used to create web pages.</p>

        <p>The <dfn></dfn> tag can be used to define a term or concept.</p>
